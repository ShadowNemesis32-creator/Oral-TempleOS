C'est quoi un OS ?

Un OS (Operating System ou système d'exploitation) est le logiciel principal qui fait fonctionner ton ordinateur, smartphone ou tablette. C'est en quelque sorte le chef d'orchestre qui :

  • Fait le lien entre le matériel (processeur, mémoire, écran, clavier...) et les applications que tu utilises
  • Gère les ressources de ton appareil (mémoire, stockage, etc.)
  • Te permet d'interagir avec ta machine via une interface graphique (fenêtres, icônes, menus)
  • Permet aux applications de fonctionner

Les exemples les plus connus sont Windows, macOS, Linux pour les ordinateurs, iOS ou Android pour les smartphones. Même les consoles de jeux vidéos ont besoin d'un OS, la PS4 utilise Orbis OS et la PS5 utilise Prospero OS.

Sans OS, ton appareil serait juste un tas de composants électroniques incapables de faire quoi que ce soit d'utile !
